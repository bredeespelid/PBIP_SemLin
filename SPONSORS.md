## FAQ

**Can I sponsor anonymously?**
Yes. On GitHub Sponsors, there is an option to make your sponsorship private. Your name will not appear anywhere public.

**Can my employer sponsor on my behalf?**
Yes. If your company routes payment through procurement, [reach out on LinkedIn](https://www.linkedin.com/in/jihwankim1975/) and I can provide the necessary details. GitHub Sponsors also supports organization-level sponsorships.

**Is this tax-deductible?**
This is not a registered charity, so sponsorships are generally not tax-deductible. Consult your local tax adviser for specifics.

**Can I make a one-time contribution instead of monthly?**
Yes — use [Buy Me a Coffee](https://buymeacoffee.com/jihwankim) for any one-time amount, or GitHub Sponsors supports one-time contributions as well.

**What happens if I cancel my sponsorship?**
Your name or logo will be removed from the README and app on the next release after cancellation. The Expert monthly session ends at the close of the current billing period. No hard feelings — life happens.

**I want a feature built. Can I pay for it?**
No. Features are not available for purchase. Sponsors at the Silver tier and above get priority on feature requests via the standard GitHub issue process, but development decisions remain mine. This keeps the tool useful for everyone.
